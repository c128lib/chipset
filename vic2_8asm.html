macro SetBasicIrqActivity ( active  )

Set Basic IRQ routine active or not.

Parameters
[in]activeActivate or deactivate basic irq
Remarks
Register .A will be modified.
Note
Use c128lib_SetBasicIrqActivity in vic2-global.asm
Turning off the BASIC IRQ routine will give you direct access to the hardware registers, you should keep in mind that it will also effectively disable the BASIC statements MOVSPR, COLLISION, SOUND and PLAY

0.6.0
